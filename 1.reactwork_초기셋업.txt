▶ npm/npx = maven/gradle 같은 api repository
   아무거나 사용가능
   npm -> 라이브러리를 다운 - 빌드 -> 프로젝트 3개(각각 프로젝트마다 로컬에 다 다운) - 프로젝트마다, 글로벌(모든 프로젝트가 바라볼 수 있게)
   npx -> 라이브러리를 다운 - 빌드 -> 프로젝트 3개(이미 다운받은게 있으면 재사용, 없으면 다운) -> 실행 -> 실행 후 삭제
